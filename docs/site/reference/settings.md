# Settings

Open Settings from the gear icon at the bottom of the left sidebar.

## Appearance

**Theme**

Brewski ships with 10 themes. Pick the one that doesn't make your eyes hurt at midnight.

| Theme | Style |
|---|---|
| Midnight | Dark, deep blue-black |
| Dracula | Dark, purple accents |
| Tokyo Night | Dark, cool blue tones |
| Catppuccin | Dark, pastel accents |
| Nord | Dark, arctic color palette |
| Monokai | Dark, warm editor-style colors |
| Catppuccin Latte | Light, pastel accents |
| Solarized Light | Light, warm tones |
| Ayu Light | Light, clean and minimal |
| GitHub Light | Light, familiar neutral tones |

Theme changes take effect immediately, no restart needed.

## Units

**Measurement System**

Switch between Metric and Imperial. This affects volume, weight, and temperature display everywhere in the app.

| System | Volume | Weight | Temperature |
|---|---|---|---|
| Metric | L | kg | °C |
| Imperial | gal | lb | °F |

**Gravity Unit**

Choose how gravity is displayed and entered across the app.

| Unit | Format | Example |
|---|---|---|
| SG | Specific gravity | 1.052 |
| Plato | Degrees Plato (°P) | 12.9 |
| Brix | Degrees Brix (°Bx) | 12.9 |

This affects recipe fields, batch readings, and all calculators.

## Ingredients

**Default Equipment Profile**

The equipment profile selected here is applied automatically when you create a new recipe. You can change the profile on any individual recipe without affecting this default.

See [Equipment profiles](/reference/equipment-profiles) for a full description of profile fields.

## Database

This section shows the current path to your Brewski database file.

**Move here**: click this to relocate the database to a new folder (for example, a cloud-synced folder like iCloud Drive or Dropbox). Brewski copies the file to the new location and updates its path.

You can also type a path directly into the field. This is useful if you want to point Brewski at an existing database you've moved manually.

For full instructions on setting up cloud sync, see [Cloud sync](/guides/cloud-sync).

## Help

**Show tooltips**

When on, small **?** icons appear next to fields throughout the app. Click them to see an explanation of what that field does. Default: on.

Turn this off once you're comfortable with the app and want a cleaner interface.
